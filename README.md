# 🏠 House Price Prediction App

A Machine Learning project that predicts house prices based on location, area, amenities, and other features using XGBoost.

## 📌 Features
- Predict house prices instantly
- User-friendly Streamlit interface
- Supports multiple cities and locations
- Handles categorical data using Label Encoding
- Uses advanced ML model (XGBoost)

---

## 🧠 Machine Learning Model
- Model: XGBoost Regressor
- Evaluation Metric:
  - R² Score: ~0.79
  - Mean Absolute Error: ~16.25
- Cross Validation Score: ~0.82
